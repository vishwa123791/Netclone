# Netclone
# Netflix Landing Page Clone

This project is a simple Netflix landing page clone built using HTML and CSS.
It focuses on layout design, fonts, and background overlay styling.

## Features
- Netflix-style hero section
- Dark background overlay
- Responsive navigation bar
- Sign In and Get Started buttons

## Technologies Used
- HTML5
- CSS3
- Google Fonts

## Folder Structure
├── index.html
├── style.css
├── assets/
│   └── images/
│       └── background.jpg

## How to Run
1. Clone or download the repository
2. Open `index.html` in a web browser
